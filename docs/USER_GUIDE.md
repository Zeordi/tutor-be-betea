# User Guide

## Parents

1. Register as a **Parent**
2. Search tutors by subject
3. Open a teacher profile and request a booking
4. Pay securely via Stripe
5. Message your tutor and leave a review after the session

## Teachers

1. Register as a **Teacher**
2. Complete your profile (subjects, rate, bio)
3. Submit verification documents
4. Set weekly availability
5. Accept bookings and track earnings

## Admins

1. Review teacher verifications
2. Manage users and bookings
3. Resolve disputes
4. Monitor analytics
